- `~/.openclaw/skills/vault-read/SKILL.md`
- `~/.openclaw/skills/vault-write/SKILL.md`
- `~/.hermes/skills/coordination/vault-handoff/SKILL.md`

如 source 更新，記得同時更新 snapshot。
