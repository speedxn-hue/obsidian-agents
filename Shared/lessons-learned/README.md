# Lessons Learned

跨 agent 共用嘅驗證過嘅經驗教訓。每次開新 task 前應該搜呢度。

## 命名

`YYYY-MM-DD-topic.md`

## 模板

```markdown
---
date: 2026-04-16
learned_by: OpenClaw | Hermes | both
applies_to: [list of topics / tools / domains]
---

# Short Lesson Title

## 情境
（當時遇到咩問題）

## 教訓
（一兩句講清楚 takeaway）

## 詳細
（點解會咁？Context）

## 點應用
（下次遇到類似情況應該點做？）
```

## 規則

- **Append-only**：唔好改舊 entry，要更新就新開一個 entry 並 link 去舊嗰個
- 呢度係 **已驗證** 嘅知識，唔係個人筆記。未證實嘅嘢應該留喺 `Agent-*/working-context/`
- 每個 entry 應該係 agent 真正可以搜得返、用得到嘅知識
